# 华叔私域运营 SOP

## 角色分工

| 执行角色 | 场景 | 核心目标 |
|---------|------|---------|
| **华叔** | 抖音直播/短视频 | 公开引流，收割信任 |
| **小助理** | 抖音私信 | 过渡承接，引导加企业微信 |
| **企业微信** | 加好友后 | 私域留存，转化付费 |
| **社群规则** | 公开群 + QQ备用群 | 承载用户，风控备份 |
| **群公告** | 企业微信群 | 每月行业方向，持续价值 |
| **群运营** | 进群后日常 | 激活用户，提升续费 |

## 整体链路

```
抖音直播/短视频（华叔）
    ↓ 引导私信扣1
抖音私信（小助理承接）
    ↓ 引导加企业微信
企业微信（留存转化）
    ↓ 付款998元/月
社群（行业方向，不讲个股）
```
